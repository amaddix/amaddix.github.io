JoiRide - RoadTrip Itinerary

Google Maps API - HTML - CSS - Java Script - EJS - Express Node.js

Collaboration project with : Greg Hairfield


We created a site with the intentions of helping users plan a road trip or vacation. The API key we used for our site is currently inactive, but when initially launched we successfully created some useful features. By entering a starting and end location, with the ability to add stops, you get direction for your route. The most useful feature of our site was the filtering tool, that allows the user to select types of places they'd like to visit along their trip (such as restaurants, trails, museums etc.).
With a time crunch our site didnt turn out the most visually appealing, there's definitely modifications I would make now given my growth since.
