Person/Vehicle Recognition Research


2022

CIFAR-10 image recognition algorithm comparison

Python Language - Machine Learning - Object Detection - AI algorithm comparison research

Team Research project : Myself along with, Forest Pearson, Ian Wasson, Eben Weisman, Jingjing Zhao


Our goal with this project was to compare the performance of 3 different algorithms tasked with classifying objects in images from the CIFAR-10 dataset. We compared results using a multi-layered perceptron, a convolutional neural network, and K-means. Ultimately we found that the unsupervised K-means algorithm preformed the poorest, which was an expected outcome, whereas the convolutional network found the best overall results.
