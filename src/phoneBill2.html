Phone Bill Manager -
Android App

Java Language - Object Oriented design - Test driven development - intelliJ - android studio - REST API - maven - javadocs - jacoco -


Through the course of an advanced java programming course, I created both a client-server web interface and an android application to simulate a phone bill managing account.

Client-Server:



This program receives information from the user on a customer and a phone call, saves that information to the server. The user can request and retrieve information about a customers phone bill history, and has the option to filter their search between two times for a customers phone bill history. With each added phone call, the program saves this new information into a local file stored on the user's smartphone, and each time the app is launched, the user has the ability to re-upload the previous data or overwrite changes.
