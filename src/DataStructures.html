Array of Linear Linked List

Array of LLL - Abstract Data Type

The program works as a sort of navigational car game. I was tasked to use a data structure of my design to hold information about a road and the possible actions to take (such as moving forward, changing lanes, possible obstructions, and traffic stops). I used an Array of linear linked lists to hold information on the road. The size of the array holds information on each lane, and the nodes along these lanes are similar to a chess style board, where the car can move from one node to the next.
