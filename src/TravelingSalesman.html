TravelingSalesman- Comparing AI algorithms

Python Language - TSP optimization - AI algorithm comparison research

Team Research project : Myself along with, Camilo Schaser-Hughes, Jason Bockover, Yufie Tao


This project focused on efficiently solving the Traveling Salesman problem, a task to find the shortest path between n locations. We analyzed and compared results from 3 different algorithms to find the most optimal method. We used the Genetic algorithm, Simulated Annealing, and an Ant Colony Optimization and found that they all preformed well when n was between 5 - 26. At n = 42 the Genetic Algorithm and ACO performed best. However, at 48, the highest value of n we tested, Simulated Annealing outperformed the rest.
